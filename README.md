# 项目简介

基于PyQt5的图像分割系统，主要针对医学及自然图像的语义分割任务。该系统采用深度学习模型CBAM-UNet，有效提升了分割的准确性和效率。功能模块包括图像预处理、模型训练、分割结果显示等，用户界面友好，操作简便。技术特点上，系统利用CBAM机制增强模型对关键特征的注意力，进而优化分割效果，适用于多种图像分割应用场景的研究与实际应用。


## 核心功能
- 图像预处理：对输入的医学或自然图像进行预处理操作
- 模型训练：基于CBAM-UNet深度学习模型的训练功能
- 分割结果显示：直观展示图像分割结果
- 友好的用户界面：基于PyQt5构建的操作简便界面

## 适用场景
- 医学图像的语义分割任务研究
- 自然图像的语义分割任务研究
- 多种图像分割应用场景的实际应用

## 技术特点
- 采用CBAM-UNet深度学习模型进行图像分割
- 集成CBAM机制增强模型对关键特征的注意力
- 提升分割准确性和效率
- 适用于医学及自然图像的语义分割任务
